# AgentSmith

Веб-приложение для общения с AI-агентом на базе Yandex GPT. Приложение имеет три режима работы:
1. **Справочник** - чат с критиком фильмов
2. **Рекомендатор** - интерактивный подбор фильмов
3. **Рассуждения** - сравнение разных способов решения задач с помощью AI

## Требования

- Python 3.7+
- API-ключ Yandex Cloud
- Catalog ID в Yandex Cloud

## Установка

1. Клонируй репозиторий:
```bash
git clone <repository-url>
cd AgentSmith
```

2. Создай виртуальное окружение:
```bash
python3 -m venv venv
source venv/bin/activate  # для macOS/Linux
# или
venv\Scripts\activate  # для Windows
```

3. Установи зависимости:
```bash
pip install -r requirements.txt
```

4. Создай файл конфигурации:
```bash
cp config.example.json config.json
```

5. Открой `config.json` и заполни своими данными:
```json
{
  "api_key": "YOUR_API_KEY_HERE",
  "catalog_id": "YOUR_CATALOG_ID_HERE"
}
```

## Запуск

```bash
python app.py
```

Приложение будет доступно по адресу: `http://localhost:5005`

## API Endpoints

- `GET /` - Главная страница с чатом
- `POST /chat` - Отправка сообщения в чат (режим Справочник)
- `POST /recommend` - Отправка сообщения агенту-рекомендатору (режим Рекомендатор)
- `POST /reasoning` - Решение задачи разными способами (режим Рассуждения)
- `POST /clear` - Очистка истории диалога
- `POST /clear_recommendations` - Очистка истории рекомендаций

## Режимы работы

### 1. Справочник
Чат с критиком фильмов, который возвращает информацию о фильмах в структурированном JSON-формате.

### 2. Рекомендатор
Интерактивный режим подбора фильмов:
- Агент задает вопросы о ваших предпочтениях
- Предлагает 16 фильмов на выбор
- После выбора 4 фильмов дает персональную рекомендацию

### 3. Рассуждения (День 4 челленджа)
Режим для изучения различных способов рассуждения AI-агентов. Позволяет решить одну задачу четырьмя разными способами и сравнить результаты:

#### Способы рассуждения:
1. **Прямой ответ** - AI отвечает напрямую, кратко и по делу
2. **Пошаговое решение** - AI решает задачу шаг за шагом, показывая всё рассуждение
3. **С промптом от ИИ** - Сначала другой AI создает оптимальный промпт для решения, затем задача решается с его помощью
4. **Группа экспертов** - Виртуальная команда из 4 экспертов (Логик, Креативщик, Критик, Синтезатор) совместно решает задачу

#### Как использовать:
1. Переключитесь на режим "Рассуждения"
2. Введите задачу (логическую, математическую или любую другую)
3. Выберите способ рассуждения или запустите все сразу
4. Сравните результаты разных подходов

#### Критерии сравнения:
- **Точность**: Правильность ответа
- **Полнота**: Насколько детально раскрыто решение
- **Понятность**: Легко ли следовать логике рассуждений
- **Креативность**: Присутствуют ли нестандартные подходы
